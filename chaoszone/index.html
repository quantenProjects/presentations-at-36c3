Wie werden aus Buchstaben diese Vektorpfade?

PaperJS is our friend



for (let offset = 0; offset < path.length;
                                        offset += delta) {
    const point = path.getLocationAt(offset).point;
    x_path.push(point.x);
    y_path.push(point.y);
}
